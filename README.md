✈️ Мини-приложение “Планировщик путешествия”
Описание:
Простой веб-сервис, где пользователь вводит город и дату поездки, а приложение показывает:
• 	Погоду
• 	Топ-3 достопримечательности
• 	Карту города
travel_planner/

│   ├── __init__.py              # Инициализация Flask-приложения

│   ├── services/
│   │   ├── weather.py           # Получение погоды через OpenWeatherMap API
│   │   └── places.py            # (опционально) Достопримечательности
